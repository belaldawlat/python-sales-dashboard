# Sales Intelligence Dashboard Docs

This folder contains the supporting documentation and visual assets for the portfolio-ready analytics project.

## Contents

- `screenshots/` — dashboard images used to showcase the UI in the project README and portfolio presentation

## Project Summary

The dashboard is structured to help users quickly understand:

- revenue performance
- product demand and ranking
- category contribution
- time-based sales trends

## Deployment Considerations

- recommended upload size: under 5 MB
- recommended row limit: under 100,000 records
- required schema: Date, Product, Category, Quantity, Price
